Le Toucan Bot

A simple discord bot that posts Le Toucan (and other ASCII art images) in a specific channel called "toucan". This is hardcoded in the text, but it wouldn't be hard to change it to another channel name (why would you want to do that though??). Configured to run on replit by default, but works on pretty much anything. It's also pretty simple to edit the buttons: just replace the relevant ASCII art variable in toucan.py and update the associated button label and you're good to go!

Features

  • Three (THREE!) buttons with their own ASCII art.
  • Automatic deployment on all linked servers when the bot starts. That's right, it runs on multiple servers at once!
  • If the toucan is praised (some variation of "praise him") in the toucan channel, the bot replies.
  • All other messages posted in the toucan channel are deleted instantly.
  • The bot also DMs the user in question and tells them to only praise le toucan (except if they're the bot's owner).
  • To avoid clutter, all messages from the bot are deleted after 5 seconds.
  • The ability to refresh the bot if things go wrong (not very often, hopefully).
  • Want to run it on Replit? Just copy the files across and enter the bot token (easy!).

Replit Setup Guide

  1. Ensure you have a channel called "toucan" (all lower case) on the server in question.
  2. Create a discord bot on the developer portal. instructions on making a bot can be found here.
  3. Get the bot token, invite the bot to the server with the "Send Messages" and "Manage Messages" bot permissions enabled.
  4. Open Replit, log in, and create a new Repl ("+" in the top right of the page).
  5. Select "import from github" and paste "github.com/Samsquamptch/LeToucan" into "GitHub URL".
  6. Once the Repl has been set up, open "Secrets" and create a new secret key with "TOKEN" as the key and the bot token as the value
  7. Press run at the top of the page and, after dependencies are installed, you now have Le Toucan Bot running!

Please note that in order to run the bot continuously on Replit without deploying, you'll have to take further steps with a keepalive function. Instructions on how to add this to a bot can be found through a quick google search and aren't very complex. You can do it, I believe in you!
